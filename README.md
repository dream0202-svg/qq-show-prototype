# 🎮 QQ秀交互原型

一个完整的移动端QQ秀交互原型，包含多个页面和丰富的交互功能。

## 🚀 在线体验

**GitHub Pages 地址**: [点击体验原型](https://你的用户名.github.io/qq-show-prototype)

> 💡 **最佳体验提示**: 建议使用手机浏览器访问，获得最佳的移动端交互体验

## 📱 功能特性

### 🎯 核心功能
- **主页面** - QQ秀角色展示和基础操作
- **装扮编辑** - 完整的装扮编辑界面，包含捏脸、装扮、动作三个标签
- **分享功能** - 多平台分享面板，支持QQ、微信、微博等社交平台
- **赠送页面** - 好友赠送QQ秀的完整流程

### ✨ 交互体验
- 🎨 **QUI组件风格** - 符合QQ产品设计规范
- 📱 **移动端优化** - 完美适配iPhone尺寸（375×812px）
- 🖱️ **触摸交互** - 支持真实的触摸反馈效果
- 🔄 **流畅动画** - 页面切换和按钮交互动画
- 📊 **真实界面** - 包含手机状态栏等细节

### 🛠️ 技术实现
- **纯HTML/CSS/JavaScript** - 无需框架，轻量级实现
- **响应式设计** - 适配不同屏幕尺寸
- **模块化交互** - 清晰的页面切换逻辑
- **优化性能** - 快速加载和流畅交互

## 📋 页面结构

```
📱 主页面 (index.html)
├── 🎨 装扮编辑页面
│   ├── 捏脸标签
│   ├── 装扮标签 (默认)
│   └── 动作标签
├── 📤 分享页面
│   ├── 好友和群组
│   ├── 社交平台
│   └── 功能操作
└── 🎁 赠送页面
    ├── QQ秀展示
    ├── 二维码分享
    └── 分享渠道
```

## 🎯 交互流程

1. **主页面** → 点击"送好友QQ秀" → **装扮编辑页面**
2. **装扮编辑页面** → 点击"赠送" → **赠送页面**
3. **主页面** → 点击"分享" → **分享页面**
4. 各页面支持返回和关闭操作

## 🛠️ 本地开发

```bash
# 克隆仓库
git clone https://github.com/你的用户名/qq-show-prototype.git

# 进入目录
cd qq-show-prototype

# 直接在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
```

## 📱 移动端体验

- **推荐浏览器**: Safari (iOS) / Chrome (Android)
- **最佳尺寸**: iPhone X/11/12 系列
- **交互方式**: 触摸点击、滑动操作
- **网络要求**: 支持离线浏览

## 🎨 设计规范

- **色彩方案**: QQ品牌蓝 (#4A90E2) + 渐变背景
- **字体**: 系统默认字体栈
- **圆角**: 统一使用12px-25px圆角
- **阴影**: 多层次阴影效果
- **动画**: 0.3s缓动过渡
